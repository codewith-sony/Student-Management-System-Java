# Student Management System (Java)

This project is developed using Java Collections Framework.

## Features
- Add student
- Display all students
- Search student by ID
- Update student marks
- Remove student by ID
- Sort students by marks

## Concepts Used
- ArrayList
- Encapsulation
- equals() and hashCode()
- Comparator
- Exception Handling
- Menu-driven program

## How to Run
Run the StudentManagementSystem.java file as a Java Application.
